Sync from Glean
Pulls pipeline & Gong data from your Glean instance
Your Glean subdomain — ask your IT team if unsure. Usually company-be.glean.com
Generate at: Glean Admin → Integrations → API Tokens. Needs Search scope.
What gets synced: Pipeline opportunities (from deal desk sheets & Salesforce records indexed in Glean) and Gong call records (titles, dates, talk ratio where available). Coaching notes and capability ratings are never overwritten.

Heads up: some browsers block direct Glean API calls from a hosted page (CORS). If the sync fails, use Import CSV instead — it covers the same data.
